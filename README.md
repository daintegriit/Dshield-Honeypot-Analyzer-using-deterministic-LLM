# CHIRON — Deterministic LLM Honeypot Intelligence Platform

CHIRON is a research-driven cybersecurity intelligence platform built on DShield honeypot telemetry, deterministic LLM reasoning, and replay-based behavioral analysis.

The system combines:
- Real-time and replayed honeypot telemetry
- Deterministic LLM-driven cyber reasoning
- Risk scoring and attack-state classification
- Interactive React-based threat intelligence dashboards
- Experiment orchestration and evaluation pipelines

## Core Features

- DShield Honeypot Analytics
- PCAP Replay Orchestration
- Governed vs Baseline LLM Evaluation
- Behavioral Threat Classification
- Risk Timeline Extraction
- Real-Time Threat Visualization
- Explainable AI Copilot Integration
- Research Artifact Generation
